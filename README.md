# DecodeLab Data Cleaning and Preparation Project

## Project Overview

This project was completed as part of my **DecodeLab Data Analytics Virtual Internship**.

The objective was to clean and prepare a raw dataset for analysis by identifying and correcting common data quality issues.

---

## Project Objectives

* Handle missing values
* Remove duplicate records
* Correct inconsistent data
* Standardize text values
* Prepare the dataset for analysis
